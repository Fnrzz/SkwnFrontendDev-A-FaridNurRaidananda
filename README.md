# Dekoor - Furniture Web App

Project ini dibangun menggunakan **React 19**, **TypeScript**, **Vite**, dan **Tailwind CSS v4**.

## 🛠️ Persyaratan Sistem (Prerequisites)

Sebelum menjalankan project, pastikan di komputer Anda sudah terinstal:

- [Node.js](https://nodejs.org/) (Versi 18 atau terbaru direkomendasikan)
- Package Manager seperti `npm`, `yarn`, atau `pnpm`

## 🚀 Cara Menjalankan Project (Development)

1. **Buka Terminal/Command Prompt**
   Pastikan Anda berada di dalam folder project ini.

2. **Install Dependencies**
   Jalankan perintah berikut untuk mengunduh semua paket yang dibutuhkan:
   ```bash
   npm install
   ```
   Jalankan Development Server
   Setelah proses instalasi selesai, jalankan perintah berikut untuk memulai server lokal:
   ```bash
   npm run dev
   ```
   Buka di Browser
   Terminal akan menampilkan URL lokal (http://localhost:5173). Buka URL tersebut di browser Anda.
